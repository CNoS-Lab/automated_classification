If a new exemplar is added, add the new mean exemplar image files to the Exemplars/orig folder, use match_sizes.m to match the exemplar size to ch2.nii, then edit the classify_component.m, order_groups.m and prepare_slices.m files ONLY. In classify_component, only change line 559 or line 624. In order_groups.m, simply add the pattern numbers to the appropriate array. In prepare_slices.m, simply add the new exemplar's name and slices, following the format in the file. See these files for more details.
